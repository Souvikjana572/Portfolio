# Souvik Jana Portfolio

A personal portfolio website for **Souvik Jana**, built to showcase software development work, technical skills, competitive programming profiles, education, professional experience, and contact information.

## Overview

This portfolio presents Souvik as a software developer, web developer, and competitive programmer. It is designed as a modern single-page experience with animated sections, interactive project showcases, coding profile links, and a contact form.

## Highlights

- Animated landing section with role typing effect and social profile links
- About section with profile summary and quick career highlights
- Scroll-based project showcase for featured work
- Skills section covering languages, web technologies, cloud, DevOps, tools, and libraries
- Professional experience section featuring Amazon internship work
- Education section with academic background
- Contact section with EmailJS-powered message form
- Responsive layout for desktop and mobile screens

## Featured Projects

### Student Management System

A role-based platform for Admin, Teacher, and Student workflows, including attendance, marks, assignments, and course management.

**Tech stack:** React, React Router, React Bootstrap, Node.js, Express, MongoDB, Mongoose, JWT, Axios

### Chess

A real-time 1v1 multiplayer chess application with live move sync for players and spectators, including game-over and reset flow.

**Tech stack:** Node.js, Express, Socket.IO, EJS, Vanilla JavaScript, chess.js, Tailwind CSS
